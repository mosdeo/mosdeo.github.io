誤差方程式

無誤差情況：Ax + By + C = z
實際上情況：Ax + By + C = f(x, y)
也就是說，存在誤差：z - f(x, y)

為了同時消除方向性、逞罰大誤差，所以將誤差平方：( z - f(x, y) )^2
將誤差方程式寫出來，求出微分=0的極值點，就能得到此方程式參數。
