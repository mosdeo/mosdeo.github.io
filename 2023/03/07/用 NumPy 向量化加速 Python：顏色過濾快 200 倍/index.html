為什麼用 np.bitwise_and.reduce()？

bitwise_and 就是取交集、reduce 就是把陣列裡面的元素做某種「化N合1」的運算。

所以這六個篩子就會化簡成一個，篩出所有符合條件的樣本。

最後可以用 img_vectorlize[skin_mask] = 0 一次對符合六個條件的樣本，改顏色。
